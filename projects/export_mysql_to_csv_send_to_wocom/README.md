# export_mysql_to_csv_send_to_wocom

Export the data in mysql into CSV files and send them to enterprise wechat group chat.

### Prerequisites

- PyMySQL==1.0.2
- requests==2.26.0

### How to run the script

```shell
# 1. edit config.ini
$ vim config.ini
# 2. run script
$ python export_mysql_data_to_csv.py
```
